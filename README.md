CODSOFT C++ Programming Internship

This repository contains the projects completed as part of the CodSoft C++ Programming Internship.
The tasks focus on developing basic console-based applications using C++ and strengthening programming fundamentals.


📌 Internship Tasks Implemented


🔢 Task 1 – Number Guessing Game

A console-based game where the program generates a random number and the user tries to guess it.

Features
	•	Random number generation
	•	User input for guessing
	•	Feedback if the guess is too high or too low
	•	Continues until the correct number is guessed

Concepts Used
	•	Random numbers
	•	Loops
	•	Conditional statements
	•	User input/output



🧮 Task 2 – Simple Calculator

A calculator program that performs basic arithmetic operations.

Features
	•	Addition
	•	Subtraction
	•	Multiplication
	•	Division
	•	User selects operation from a menu

Concepts Used
	•	Functions
	•	Switch/conditional statements
	•	Arithmetic operators
	•	Console input/output



📚 Task 5 – Library Management System

A console-based system to manage books in a library.

Features
	•	Add new books
	•	View book list
	•	Store book details (title, author, ID)
	•	Manage library records

Concepts Used
	•	Structures (struct)
	•	Vectors for storing data
	•	Functions
	•	Menu-driven programs

🛠 Technologies Used
	•	C++
	•	VS Code
	•	Git & GitHub
